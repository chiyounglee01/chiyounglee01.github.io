(It may take a minute for the model to load.) <br></br><h1><b>Overview</b></h1><br></br><p>The goal of this project was to train (fine-tune) a Llama-3.1-8B model on existing Seinfeld monologues to make an LLM which can output a Seinfeld style monologue by inputing a keyword. The project used <b>Python</b> code.</p><p>The training text was obtained by scraping, extracting, and cleaning scripts from the "Seinfeld Scripts" <a href="http://seinfeldscripts.com" target="blank">website</a>.</p><p>The tools used were - Python, Torch, Llama-3.1-8B model, Unsloth, LoRA Adapters (for efficient training), Gradio (for the LLM UI), Hugging Face (LLM hosting).</p>
